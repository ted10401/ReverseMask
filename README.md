# Reverse Mask Component

## How to Use
* Create a Canvas
* Set the Render Mode to Screen Space - Camera
* Set the camera to Render Camera
* Add Reverse Mask Layer
* Set Reverse Mask Layer Color
* Add Reverse Mask
* Set the mask sprite to ReverseMask::Source Image
* Finish
